# Hexo博客管理
基于C++/QML编写hexo博客管理程序，仅支持导入用GithubPages托管的博客

---

## 🔨 构建

最低版本要求

> Qt 6.5
>
> CMake 3.16
>
> C++17

Qt Creator选中`CMakeLists.txt`导入，选择MSVC 64bit即可

---

## 🌲 特性

- 🌟 **深色/浅色模式无缝切换**
- 🪐 **主题颜色支持自定义选择**
- 🌈 **响应式布局，自适应窗口尺寸变化**
- 🎈 **支持编辑和预览markdown文件**
- 🎁 **利用ListView实现可视化博客提交记录**
- 🧩 **快捷编辑Hexo和主题的YAML配置文件**
