# Hexo博客管理
基于C++/QML编写hexo博客管理程序，仅支持导入用GithubPages托管的博客

---

## 🔨 构建

最低版本要求

> Qt 6.5.3
>
> CMake 3.16
>
> C++17

Qt Creator可以直接选中`CMakeLists.txt`导入，选择MinGW 64即可（已支持MSVC 64）

---

## 🌲 特性

- 🌟 **深色/浅色模式无缝切换**
- 🪐 **主题颜色支持自定义选择**
- 🌈 **响应式布局，自适应窗口尺寸变化**
- 🎈 **支持编辑和预览markdown文件**
- 🎁 **利用QCharts图表可视化博客提交记录**
